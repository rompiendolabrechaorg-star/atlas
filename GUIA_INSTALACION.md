# 🚀 Guía de Inicio: ¿Cómo usar Atlas de Ideas?

Si quieres ejecutar este programa en tu ordenador y no sabes nada de programación, ¡no te preocupes! Sigue estos pasos sencillos.

## 1. Preparativos (Solo la primera vez)

Para que el programa funcione, necesitas instalar una herramienta llamada **Node.js**. Es como el "motor" que permite que la aplicación arranque.

1.  Ve a [nodejs.org](https://nodejs.org/).
2.  Descarga la versión que dice **"LTS"** (es la más estable).
3.  Instálalo en tu ordenador dándole a "Siguiente" en todo.

---

## 2. Descargar el Proyecto

1.  Entra en el repositorio de GitHub: [rompiendolabrechaorg-star/atlas](https://github.com/rompiendolabrechaorg-star/atlas).
2.  Busca el botón verde que dice **"<> Code"**.
3.  Haz clic en **"Download ZIP"**.
4.  Descomprime esa carpeta en tu escritorio o donde prefieras.

---

## 3. Arrancar el Programa

Ahora vamos a "darle vida" al código:

1.  Abre la carpeta que has descomprimido.
2.  Haz clic derecho en un espacio vacío de la carpeta y elige **"Abrir en la Terminal"** (o "Abrir ventana de comandos").
3.  Escribe el siguiente comando y pulsa Enter:
    ```bash
    npm install
    ```
    *(Esto descargará las piezas necesarias para que el programa funcione. Tardará un minuto).*
4.  Cuando termine, escribe este comando y pulsa Enter:
    ```bash
    npm run dev
    ```
5.  Verás que pone algo como `http://localhost:5173`. **¡Haz clic en ese enlace!** Se abrirá el programa en tu navegador.

---

## 4. Configurar la "Llave Inteligente" (IA)

Para que la Inteligencia Artificial (Gemini) funcione:

1.  En la pantalla principal, haz clic en el icono del **engranaje (⚙️)** arriba a la derecha.
2.  Pega tu **API Key de Gemini**.
    *   *Si no tienes una, haz clic en el enlace de "Google AI Studio" que verás allí mismo o pídela al administrador.*
3.  Dale a **Guardar**. ¡Ya puedes usar todas las funciones de IA!

---

> [!TIP]
> **Forma Súper Fácil (Recomendada):** 
> Si no quieres usar la terminal, puedes conectar este repositorio a **Vercel** o **Netlify**. Son plataformas gratuitas donde subes el código y te dan un enlace web real (como `mi-proyecto.vercel.app`) que funciona en cualquier móvil o tablet sin instalar nada.
